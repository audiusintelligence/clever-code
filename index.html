clever code

Eine Clever Solution in 5 Minuten — direkt vom Mac, ohne Repo-Klon.

Installation (einmalig)

# 1. Docker
brew install --cask docker

# 2. opencode
curl -fsSL https://opencode.ai/install | bash

# 3. Clever Toolchain
curl -fsSL https://code.clevercompany.ai/install | bash

# 4. Keycloak Login
clever auth

Erste Solution erstellen

clever new invoice-tracker
# oder
opencode "neue Clever Solution namens invoice-tracker für Rechnungen"

opencode liest die Guides, generiert frischen Code, legt einen Keycloak-Client an, startet den Stack. Nach 2 Minuten ist http://localhost:7100 live.

Was wird installiert?

PfadInhalt
~/.clever/guides/Brand, Architecture, Conventions, Recipes
~/.clever/scripts/Keycloak Helper, clever CLI
~/.clever/solutions/Deine erstellten Solutions
~/.config/opencode/agent/clever-solution Agent

Befehle

CommandWas
clever authKeycloak Admin Login (einmalig pro 5min)
clever new <name>Neue Solution (ruft opencode)
clever invite <name> <email>User freischalten
clever listEigene Solutions zeigen
clever deploy <name>Auf Production deployen
clever updateGuides + Scripts aktualisieren

Wie funktioniert das?

  1. opencode ist die UX — fragt was du willst, erklärt was passiert
  2. Guides in ~/.clever/guides/ sind die Quelle für Branding, Architektur, Code-Style
  3. Bash Scripts machen nur die deterministischen Sachen (Keycloak API, Ports, Deploy)
  4. Der Code wird generiert, nicht aus einem Template kopiert. Das heißt: jede Solution ist auf den Use Case zugeschnitten.
